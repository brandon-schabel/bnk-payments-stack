# htmlody-template

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Deploy with server:
If you haven't already

```bash
brew install flyctl
```

[Docs]("https://fly.io/docs/hands-on/install-flyctl/")

## If you haven't signed up with fly

```bash
fly auth signup
```
